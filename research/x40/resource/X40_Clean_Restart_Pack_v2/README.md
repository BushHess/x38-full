# X40 Clean Restart Pack v2

This pack is the corrected, implementation-ready rewrite of the x40 operating layer.

It exists to do four things cleanly:

1. qualify one active baseline per league,
2. measure durability, decay, crowding and drift,
3. adjudicate formal challengers without premature auto-promotion,
4. choose the next research branch without mixing leagues, cost regimes or verdict namespaces.

This v2 pack supersedes the earlier clean-restart v1 pack because it fixes four load-bearing issues:

- explicit **apples-to-apples comparison discipline**,
- actual **template files** instead of template promises,
- corrected **challenger routing namespaces**,
- and consistent **registry / artifact conventions**.
