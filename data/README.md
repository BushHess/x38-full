# Data Directory

All market data files for btc-spot-dev.

## Multi-Resolution (primary — used by framework)

| File | Content | Date Range | Size |
|------|---------|------------|------|
| `bars_btcusdt_2016_now_h1_4h_1d.csv` | H1+H4+D1 combined | 2017-08 → 2026-02 | 13 MB |
| `bars_btcusdt_2017_now_15m.csv` | 15m only | 2017 → 2026 | 38 MB |

`bars_btcusdt_2016_now_h1_4h_1d.csv` is the **primary data file** used by:
- Validation pipeline (`validation/cli.py`)
- BacktestEngine (`v10/core/data.py`)
- All research studies (`research/x0/` through `research/x35/`)

## Single-Resolution (per-timeframe splits)

| File | Interval | Source |
|------|----------|--------|
| `btcusdt_15m.csv` | 15m | data-pipeline |
| `btcusdt_1h.csv` | 1h | data-pipeline |
| `btcusdt_4h.csv` | 4h | data-pipeline |
| `btcusdt_1d.csv` | 1d | data-pipeline |

## Data Pipeline

Generated by `/var/www/trading-bots/data-pipeline/fetch_binance_klines.py`.
Output path configurable via `--out` argument.

## Schema

All CSVs share the same schema:
```
symbol, interval, open_time, close_time, open, high, low, close,
volume, quote_volume, num_trades, taker_buy_base_vol, taker_buy_quote_vol
```
